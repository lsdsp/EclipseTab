## ✨ 产品简介

Eclipse Tab 将你的浏览器新标签页变成一个强大的工作台：

- ✏️ **Zen Shelf（灵感白板）** - 随时随地记录灵感，支持文字和图片贴纸
- 🌐 **Focus Spaces（多重空间）** - 为不同场景创建独立工作空间
- 🚀 **Dock 应用栏** - macOS 风格的应用管理，优雅高效
- 🔍 **智能搜索** - 多引擎支持，实时搜索建议
- 🎨 **精美主题** - 多种主题模式，自定义壁纸

## 🎯 核心功能

### Zen Shelf - 灵感白板

**你的创意画布，随时捕捉灵感**

Zen Shelf 将新标签页变成一个自由的创意空间，就像桌面上的便签纸和照片墙，让灵感随处可贴。

#### 📝 文字贴纸

**快速记录想法**
- 双击页面空白处，立即创建文字贴纸
- 右键菜单也可快速添加

**自由定制样式**
- 选择喜欢的字体颜色
- 调整文字对齐方式（左对齐、居中、右对齐）
- 滑动调节字号大小

**灵活编辑**
- 右键点击贴纸即可编辑
- 修改时保留原有样式
- 一键导出为图片保存

#### 🖼️ 图片贴纸

**多种添加方式**
- 右键菜单上传本地图片
- 直接粘贴剪贴板图片（Ctrl+V）
- 拖拽图片文件到页面

**便捷操作**
- 鼠标滚轮缩放图片大小
- 复制图片到剪贴板
- 导出为 PNG 文件

#### 🎨 自然交互

**拖拽体验**
- 点击拖动贴纸到任意位置
- 拖拽时带有轻微旋转效果，模拟真实纸张
- 文字贴纸靠近边缘会自动换行

**层级管理**
- 双击贴纸置顶显示
- 多个贴纸自动管理层级

**创意模式**
- 右键菜单开启创意模式
- 隐藏所有界面元素，专注创作
- 获得纯净的创作空间

#### ⌨️ 快捷操作

- `双击背景` - 快速添加文字贴纸
- `Ctrl+V` - 粘贴图片
- `Delete / Backspace` - 删除选中的贴纸
- `鼠标滚轮` - 缩放图片贴纸
- `右键` - 打开功能菜单

### Focus Spaces - 多重空间

**场景化工作空间，一键切换专注状态**

Focus Spaces 让你为不同场景创建独立的工作空间，每个空间有自己的应用列表，真正实现工作、学习、娱乐的完美分离。

#### 🌐 多空间管理

**轻松创建空间**
- 右键点击 Dock 右侧的 Navigator 按钮
- 选择"Add space"创建新空间
- 自定义空间名称

**灵活管理**
- 重命名空间，清晰标识不同场景
- 删除不需要的空间
- 置顶常用空间，快速访问

#### 🔄 快速切换

**一键切换**
- 点击 Navigator 按钮循环切换空间
- 平滑动画过渡，视觉体验流畅

**动画效果**
- 当前空间图标优雅退场
- 新空间图标依次入场
- Dock 宽度自动适应

#### 📦 导入导出

**备份和分享**
- 导出当前空间为 JSON 文件
- 包含所有应用和配置
- 导入他人分享的空间配置

#### 🎯 场景化应用

**工作空间**
- 邮箱、项目管理工具
- 开发工具、文档平台
- 团队协作应用

**学习空间**
- 在线课程、笔记应用
- 学习资源、词典工具
- 文档阅读器

**娱乐空间**
- 视频网站、音乐平台
- 社交媒体、游戏网站
- 新闻资讯

## 📦 安装使用

### 浏览器扩展安装

**Chrome / Edge**
1. 下载项目
2. 打开 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目中的 `dist` 文件夹

**Firefox**
1. 从 [Releases](../../releases) 页面下载 `.xpi` 文件
2. 打开 Firefox，将 `.xpi` 文件拖入浏览器窗口
3. 点击"添加"确认安装扩展

### 开始使用

安装完成后，打开新标签页即可开始使用：

1. **添加应用** - 点击右上角编辑按钮，添加常用网站
2. **创建空间** - 右键点击 Navigator，创建不同场景的空间
3. **记录灵感** - 双击页面添加文字贴纸，或粘贴图片
4. **个性化主题** - 点击左上角设置，选择喜欢的主题

## 💡 使用技巧

### Zen Shelf 技巧

- 使用创意模式获得纯净的创作空间
- 文字贴纸支持导出为图片，方便分享
- 图片贴纸可以直接复制到剪贴板
- 双击贴纸可以置顶显示

### Focus Spaces 技巧

- 为工作、学习、娱乐创建不同空间
- 使用导出功能备份重要空间配置
- 置顶常用空间，提高切换效率
- 每个空间独立管理应用，互不干扰

### Dock 技巧

- 长按图标快速进入编辑模式
- 拖拽应用到应用上创建文件夹
- 文件夹少于 2 个应用时自动解散
- 悬停在文件夹上 500ms 自动打开

## 📝 关于项目

Eclipse Tab 是一个使用现代 Web 技术构建的浏览器扩展项目，90% 的代码通过 AI 辅助编码（VibeCoding）完成。

**技术栈**
- React 18 + TypeScript
- Vite 构建工具
- CSS Modules

**数据存储**
- 所有数据自动保存到本地
- 使用 localStorage 和 IndexedDB
- 支持大容量壁纸存储（突破 5MB 限制）
